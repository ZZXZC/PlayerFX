# Excel Audio Timestamp Player 🎵

A desktop application built with JavaFX and Apache POI that parses timestamps from Excel spreadsheets to trigger specific audio clip segments.

##  Key Features
- **Spreadsheet Reader:** Automatically parses `.xlsx` files and validates timestamp data.
- **Media Controls:** Built-in audio playback engine with precise timestamp jumping.
- **Standalone Executable:** Packaged as a native `.exe` — no Java runtime required to install.
